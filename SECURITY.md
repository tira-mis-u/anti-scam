# Security Policy

## Supported Versions

| Version | Supported |
|----------|----------|
| 1.x | ✅ |

## Reporting a Vulnerability

Nếu bạn phát hiện lỗ hổng bảo mật trong AntiScam:

1. Không công khai lỗ hổng trên Issue.
2. Liên hệ maintainer thông qua:
   - GitHub Security Advisory
   - Email: trisnosleep@example.com

## Scope

Chính sách này áp dụng cho:

- Browser Extension
- Desktop Application
- Backend API
- Website Dashboard

## Disclosure Policy

Chúng tôi khuyến khích Responsible Disclosure (công bố có trách nhiệm).
Vui lòng không công khai lỗ hổng trước khi bản vá được phát hành.
